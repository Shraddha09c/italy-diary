# Italy Diary

Italy Diary is a simple and responsive image gallery built using HTML and CSS. It showcases some of the most beautiful and iconic places to visit in Italy with elegant hover effects and a clean card-based layout.

## 📸 Preview

A travel-inspired gallery featuring:
- Colosseum
- Leaning Tower of Pisa
- Trevi Fountain
- Venice Grand Canal
- Milan Cathedral
- Cinque Terre
- Amalfi Coast
- Florence Cathedral
- Lake Como

## 🚀 Features

- Responsive Grid Layout
- Smooth Hover Animation
- Grayscale to Color Transition
- Image Zoom Effect
- Gradient Caption Overlay
- Clean and Modern UI
- Beginner-Friendly HTML & CSS Project

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox Concepts
- CSS Transitions
- CSS Transform
- CSS Filter
